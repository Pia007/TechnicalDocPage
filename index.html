Introduction

Cascading Style Sheets, CSS, is a style sheet language that specifies how documents are presented to users. The documents are text files structured using Hyper Text Markup Language, HTML.

Objectives:

  • Understand the basics of linking a CSS document to and HTML file.
  • Apply simple text formating using CSS.

Prerequisites

  • Have basic computer literacy.
  • Have basic software installed.
  • Have basic knoweledge of working with files.
  • Understand HTML basics.

Adding CSS to a Document

The HTML document needs to know that there are CSS rules available before they can be applied.

There are three different ways to apply CSS.

  • Inline
  • Internal
  • External

For introductory pruposes, the most common and useful way is to link CSS in the head of the HTML document.

Create a folder in the same folder as your HTML document, index.html, and save it as styles.css. The .css extension shows that the file is a CSS file.

Add the line of code below somewhere inside the <head> of the HTML document to link styles.css to index.html.

<link rel="stylesheet" href="styles.css">

Each part of the code has a role in informing the browser.

  • <link> element tells the broswer that there is an external source, the CSS file.
  • rel attribute specifies the relationship between the document and the linked resource.
  • href attribute specifies the link's location.

Use your code editor to add the level one heading declaration below to your CSS file:

               h1 {
                  color: red; 
               }

Save your HTML and CSS files and reload the page in a web browser. The level one heading at the top of the document should be red. If its not, check for accuracy.

Styling HTML Elements

Making the heading red demonstrated that an HTML element can be targeted and styled by using an element selector, a selector that selects all elements with the matching name.

Use the selector p to turn all paragraphs in the document green.

  
              p {
                color: green;
              }

Target multiple selectors, by using a comma to seperate the selectors. The rule below turns all paragraphs and list items green.

  
              p, li {
                color: green;
              }
Change the Behavior of Elements

All modern browsers have internal stylesheets that contain default styles. These styles apply to all of the pages by default. They prevent the text from running together, resulting in having to style everything from scratch.

A good example of a browser' default styling is an unordered list, <ul>. By default the list items, <li> are bulleted.

Try adding the code below to your CSS.

  
              li {
                list-style-type: none;
              }

One of the codes below takes a different approach to bulletting. Try them:

list-style-type: circle; list-style-type: "\1F44D";

In addition to a circle bullet point, you should have gotten a "thumbs-up". Check the MDN for more information about the list-style-property.

Adding a Class

In your, HTML document, add a class element to the second list item. The code should look like this:

              <ul>
                  <li>Item one</li>
                  <li class="special">Item two</li>
                  <li>Item<em>three</em></li>
              </ul>

In your CSS you can target the class of special by creating a selector that starts with a full stop character.

Try adding the following code to your CSS file.

              .special {
                color: orange;
                font-weight: bold;
              }

The class of special can be added to any element that you want to have the same look as the list item.

In the code below, the rules have a selector that lists the HTML element selector along with the class. According to the rule below, any li element that has a class of special will be styled orange with bold font. 

              li.special {
                color: orange;
                font-weight: bold;
              }

After including the class with the HTML element selector, the class can no longer be added to an another element. The class cannot be added to another element unless element is added to the list of selectors. For example, in the CSS rule below, the class of special was listed along with the li element. The <span> element is added to the list of selectors in order to apply the orange color and bold font.

              li.special,
              span.special {
                color: orange;
                font-weight: bold;
              }
Styling Based on Location

There are occassions when something is styled differntly based on its location in a document. Ther several selectors that can help accomplish this. In the initial HTML code, two <em> were used. One inside the paragraph and the other inside a list item. To select only the <em> nested inside a <li>, a selector called the descendant combinator, a space between two selectors is used.

Add the following rule to your stylesheet:

                .li em {
                  color: rebeccapurple;
                }

In the document, only the <em> in the third list item turned purple.

Additionally, a paragraph can be styled when it comes directly after a heading at the same hierarchy level in HTML by using an adjacent sibling combinator, a + between the selectors.

Add the following rule to your stylesheet:

                h1 + p {
                  font-size: 200%;
                }

The font sizes of header one and the paragraph immediately are twice as large as they were originally.

Styling Based on State

The last type of styling for this tutorial is styling things by their state. The best way to describe this type of styling is when styling links. To style a link, the <a>(anchor) element is targeted. The link has different states based on whether it is unvisited, visited, being hovered over, focused via the keyboard, or in the process of being clicked (activated).

The CSS below styles unvisited links pink and visited links green.

                a:link  {
                  color: pink;
                }
      
                a:visited {
                  color: green;
                }

The way the link looks when the user hovers over it, for example removing the underline, can be achieved by the rule below:

                a:hover {
                  text-decoration: none;
                }

Try replacing the value of the text-decoration property with the values below:

text-decoration: underline; text-decoration: underline dotted red;
Join Selectors and Combinators

Multiple selectors and combinators can be combined together. For example:

 
                /* selects any <span> that is inside a <p>, which is inside an <article> */
                article p span { ... }
      
                /* selects any <p> that comes directly after a <ul>, which comes directly after an <h1> */
                h1 + ul + p { ... }

Multiple types can be combined together too. Try the following code:

                body h1 + p .special {
                  color: yellow;
                  bagkground-color: black;
                  padding: 5px;
                }

The code above, will sytle any element with the class of special, which is inside a p, which comes just after an <h1>, which is inside a <body>. Thus, in the original HTML, the only element styled is <span class="special">.
